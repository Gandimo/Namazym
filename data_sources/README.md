# Namazym Raw Data Sources

This directory contains the original raw data used to generate the application's JSON assets.

## Structure
- `atlar/`: Original Quran verse texts and source files.
- `quran_ar_full.json`: Generated 6236 verses in Arabic (Uthmani).
- `quran_tm_full.json`: Generated 6236 verses in Turkmen.

**Note:** All data is processed into `src/namazym/data/` for use in the mobile application. These files are kept for traceability and regeneration if needed.
