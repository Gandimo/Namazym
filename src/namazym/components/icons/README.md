# Premium Icon System

## Philosophy
**Ultra-Minimal + Quiet Luxury**

This icon system is designed to provide a calm, spiritual, and high-end aesthetic. It avoids the visual noise of emojis and the generic feel of Material Design.

## Specifications
- **Stroke**: 1.5px (Monoline)
- **Caps/Joins**: Round
- **Color**: `#C4A050` (Soft Gold)
- **Size**: 24px base

## Usage

```tsx
import { StarIcon, MosqueIcon } from './icons';

<StarIcon />
<MosqueIcon size={32} color="#7A5A12" />
```
